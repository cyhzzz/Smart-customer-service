# 智能客服系统 - 产品原型

> 源于某股份制银行APP智能客服功能模块的首个原型稿，2017年设计并推进上线

## 📱 项目简介

本项目是一个智能客服系统的产品原型展示页，采用现代化设计风格，展示了从行业调研到产品上线的完整设计思路。

### 🎯 核心特点

- **苹果风格设计** - SF Pro 字体 + 黑白灰配色 + 精致细节
- **样机展示** - 真实的手机外壳展示效果
- **样式切换** - 支持现代样式和原始样式切换
- **响应式布局** - 16:9 横向布局，左侧样机 + 右侧说明
- **流畅动画** - fadeInUp 入场动画 + 悬停效果

## 🎨 设计特点

### 1. 设计思路

**行业调研** → **大数据分析** → **功能设计**

- **宫格功能**：将可功能化的高频需求开发为新功能快捷入口
- **我关心的**：零散热点问题按类别划分展示，提供个性化推荐
- **自助工具**：常用功能自助化，减少人工咨询压力

### 2. 产品上线效果

✅ **行业跟进** - 卡号查询等功能通过合规审核，获行业跟进  
✅ **咨询减少** - 客服进线涉及已上线功能的咨询显著减少  
✅ **体验提升** - 提升零售客户服务咨询体验  
✅ **降本增效** - 助力客服中心降本增效效果明显  

### 3. 视觉设计

- **字体**：SF Pro Display + SF Pro Text（苹果官方字体）
- **配色**：黑白灰 + 蓝色点缀
- **布局**：大量留白，清晰的层次
- **动画**：流畅、微妙、自然
- **细节**：精致阴影、圆角、磨砂玻璃效果

## 🛠️ 技术栈

- **HTML5** - 语义化标签
- **CSS3** - 现代化样式（Grid、Flexbox、CSS Variables）
- **Tailwind CSS** - 实用优先的 CSS 框架
- **Font Awesome** - 图标库
- **Google Fonts** - SF Pro Display 字体
