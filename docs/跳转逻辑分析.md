## 一、分析策略

### 1.1 原始页面结构
- **文件类型**: Axure 导出的 HTML（包含大量 JavaScript）
- **跳转方式**:
  - JavaScript 事件（onclick）
  - 链接跳转（href）
  - 动态加载（iframe）
- **数据文件**: `files/[页面名]/data.js`（包含跳转逻辑）

### 1.2 提取方法
```bash
# 查找所有跳转链接
grep -r "location.href\|window.location\|parent.location" *.html

# 查找所有 data.js 文件中的跳转
find files -name "data.js" -exec grep -l "onClick\|onClick:" {} \;
```

## 四、跳转逻辑实现方案

### 4.1 现代版本跳转方式

**方案1**: 直接链接（推荐）
```html
<a href="公告.html" class="btn-active">公告</a>
```

**方案2**: JavaScript 跳转
```html
<div onclick="window.location.href='公告.html'">公告</div>
```

**方案3**: 相对路径
```html
<!-- 在 modern/ 目录下的页面 -->
<a href="公告.html">公告</a>  <!-- 同目录 -->
<a href="../智能客服首页.html">首页</a>  <!-- 上级目录 -->
```
