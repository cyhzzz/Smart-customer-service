## 技术实现

### 修改方式
将所有 `onclick="showToast('...')"` 替换为 `onclick="window.location.href='目标页面.html'"`

### 示例代码

**修复前：**
```html
<button onclick="showToast('正在加载开户行查询...')" class="...">
```

**修复后：**
```html
<button onclick="window.location.href='开户行查询.html'" class="...">
```
